## Add flask environment variables:

- Windows
  - set FLASK_ENV=development
  - set FLASK_APP=<NAME-OF-PROJECT>(ie drone_inventory)
- Mac
  - export FLASK_ENV=development
  - export FLASK_APP=<NAME-OF-PROJECT>(ie drone_inventory)
